# simple_filemanager
A simple Node.js application that performs file operations such as create, read, update, and delete using the fs module.

📌 Features

Create new files

Read file content

Update existing files

Delete files

Simple UI using EJS

Server built with Node.js

🛠️ Technologies Used

Node.js

Express.js

EJS (Embedded JavaScript Templates)

File System Module (fs)

HTML / CSS

📖 How It Works

The UI is created using EJS templates.

Users can perform file operations through the interface.

Node.js handles file operations using the fs module.

Express.js manages routing between pages.

🎯 Purpose

This project is created to understand:

File handling in Node.js

Basic backend development

CRUD operations on files
